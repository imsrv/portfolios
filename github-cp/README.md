## Chartered Accountant Portfolio

A responsive, high-end single-page portfolio for a Chartered Accountant featuring modern design with deep blues and slate grays.

### Features

- **Hero Section**: Eye-catching gradient background with clear call-to-action
- **Expertise Section**: Three service cards highlighting Tax, Audit, and Advisory services
- **About/Credentials**: Professional background and qualifications display
- **Process Timeline**: Visual 4-step process workflow
- **Inquiry Form**: WhatsApp integration with form data pre-filled in chat

### Usage

Simply open `index.html` in a web browser. The portfolio is fully responsive and works on all devices.

### WhatsApp Integration

The inquiry form collects:
- Name
- Service Required
- Message

Upon submission, it redirects to WhatsApp (919876543210) with all form data structured as a pre-filled message.
